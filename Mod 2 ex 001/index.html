Como criar um álbum de fotos usando HTML5 e CSS3. Nessa aula, o professor Gustavo Guanabara (ou seria Optimus Prime?) vai mostrar uma maneira bem criativa de criar um álbum de fotos utilizando apenas listas e folhas de estilo. Utilizando as tags UL e LI, criaremos listas em HTML5. Cada um dos LI serão utilizados para carregar fotos. Utilizando a tag SPAN, criaremos legendas interativas para as fotos.
